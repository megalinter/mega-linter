# dart Mega-Linter Flavor

## Description

Mega-Linter optimized for DART based projects

## Usage

- With GitHub Action: **nvuillam/mega-linter/flavors/dart@v4**
- With Docker image: **nvuillam/mega-linter-dart:v4**
