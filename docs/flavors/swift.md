# swift MegaLinter Flavor

![Docker Image Size (tag)](https://img.shields.io/docker/image-size/megalinter/megalinter-swift/v6-alpha)
![Docker Pulls](https://img.shields.io/docker/pulls/megalinter/megalinter-swift)

## Description

Optimized for SWIFT based projects

## Usage

- [GitHub Action](https://megalinter.github.io/v6-alpha/installation/#github-action): **megalinter/megalinter/flavors/swift@v6-alpha**
- Docker image: **megalinter/megalinter-swift:v6-alpha**
- [mega-linter-runner](https://megalinter.github.io/v6-alpha/mega-linter-runner/): `mega-linter-runner --flavor swift`
